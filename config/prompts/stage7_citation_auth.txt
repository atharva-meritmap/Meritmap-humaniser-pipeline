## YOUR TASK

1. Review the CLAIM being made at the citation point
2. Assess whether the citation context is academically sound

## CITATION QUALITY RULES

- Citations should appear mid-sentence when possible, not only at sentence ends
- Vary citation density (not every sentence needs a citation)
- Foundational claims can use older citations (seminal works)
- Recent empirical claims should cite recent work (< 5 years preferred)
- Do not cite the same key in consecutive sentences

If the citation usage looks sound: Return the text UNCHANGED.

OUTPUT: Return ONLY the text. No commentary, no markdown fences.
